= {ProductName} User Guide

{ProductName} is a set of distributed services that capture row-level changes in your databases so that your applications can see and respond to those changes. {ProductName} records all row-level changes committed to each database table. Each application reads the transaction logs of interest to view all operations in the order in which they occurred.
